CS3113HW3 - Space Invaders
=========

CS3113HW3

  *Make Space Invaders!

  *It must have 3 states: TITLE SCREEN, GAME, and GAME OVER

  *It must keep and display score

  *It must use sprite sheets

  *You can use any graphics you want (it doesn't have to be in space!) :)

  *Organize your code!!
